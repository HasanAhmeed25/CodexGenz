# CodexGenz Website

Modern, responsive, glassmorphism landing page for CodexGenz.

## Files

- `index.html` - hero, about, services, learning, features, testimonials, footer.
- `styles.css` - dark glassmorphism theme, responsive layout, animations.
- `script.js` - theme toggle, mobile menu, scroll reveal.

## Usage

Open `index.html` in your browser to preview the site.
